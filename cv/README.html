Data

The data for the CV lives in an XML file. An associated XSL file describes the translation from XML to HTML. If you have scripting enabled in a modern browser, it will automatically apply the translation and display the resulting (rendered) HTML document. Finally, a stylesheet controls final formatting.

Additionly, a second XSL document provides the translation to LaTeX, from which a PDF is made for distribution.
